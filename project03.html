Finding Website Vulnerabilities

This project demonstrates a Broken Access Control attack and shows how an attacker can find and use small mistakes in a website to take total control of a computer server. It starts by showing how simple information hidden in a web browser can be used to "trick" a website into giving up secret access. From there, it shows how a common coding error allows an outsider to send their own commands to the server, much like a remote control. Finally, the project illustrates how a person can move through hidden folders to find saved passwords and eventually gain the highest level of permission, proving why every layer of a system needs to be properly locked down.
